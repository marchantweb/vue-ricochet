# Config

## Built-in layouts

### Line

Draws the elements in a straight line. The configuration for a line only requires a start and end point, which looks like:

```json
{
    "start": {
      "x": 50, 
      "y": 50
    },
    "end": {
      "x": 300,
      "y": 300
    }
}
```
